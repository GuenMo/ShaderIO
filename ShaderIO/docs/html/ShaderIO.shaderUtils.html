ShaderIO.shaderUtils module¶

ShaderIO.shaderUtils.assignShader()[source]¶

씬안에서 shaderInfoNode 노드를 검색 한다.

shaderInfoNode 노드에 shaderInfos# 어트리뷰트를 검색한다.

shaderInfos# 어트리뷰트의 값을 json 으로 가져 온다.

어트리뷰트의 geometry, surfaceShader, displacement 키 값을 가져 온다.

geometry에 surfaceShader, displacement 적용 한다.

shaderInfoNode 삭제 한다.

Warning

씬안에 ‘shaderInfoNode’ 노드가 없을 때 shaderInfoNode not exist! 예외 발생
